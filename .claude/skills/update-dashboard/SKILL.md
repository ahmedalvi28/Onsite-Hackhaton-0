# Update Dashboard Skill

## Description
Creates or updates a dashboard showing the current state of all tasks and the AI employee system.

## Usage
Use this skill when:
- You need to see the current status of all tasks
- After processing tasks or executing plans
- Periodic status updates

## Parameters
- `vault_path`: Path to Obsidian vault (default: C:/Users/alvia/OneDrive/Desktop/AI_Employee_Vault_Hackathon)

## Expected Actions

1. **Count all tasks** in each folder:
   - Drop (incoming items)
   - Inbox (new arrivals)
   - Needs_Action (pending processing)
   - In_Progress (active work)
   - Pending_Approval (awaiting review)
   - Approved (ready to execute)
   - Rejected (cancelled)
   - Done (completed)

2. **Count by category**:
   - Business tasks
   - Personal tasks
   - Other

3. **Count by priority**:
   - High priority
   - Medium priority
   - Low priority

4. **Generate dashboard** with visualization
