# Create Plan Skill

## Description
Creates a detailed Plan.md file for a specific task from the Needs_Action folder.

## Usage
Use this skill when:
- You have a specific task file that needs a plan
- A watcher (Gmail, LinkedIn, etc.) created a new task
- You need to break down a task into actionable steps

## Parameters
- `task_file`: Path to the task file in Needs_Action folder
- `vault_path`: Path to Obsidian vault (default: C:/Users/alvia/OneDrive/Desktop/AI_Employee_Vault_Hackathon)

## Expected Actions

1. **Read the task file** to understand:
   - Task type (file_drop, email, linkedin_notification, etc.)
   - Category (business/personal)
   - Priority (high/medium/low)
   - Content and requirements

2. **Generate a comprehensive plan** including:
   - Task breakdown into steps
   - Required resources
   - Estimated complexity
   - Approval requirements

3. **Create Plan.md** in Plans folder
4. **Move task** to In_Progress folder
5. **Log the action**

## Plan.md Format

```markdown
---
task_id: TASK_ID
task_type: TASK_TYPE
status: pending_approval
priority: PRIORITY
category: CATEGORY
created: TIMESTAMP
---

# Plan: [Descriptive Title]

## Task Summary
[2-3 sentence summary of what this task accomplishes]

## Background/Context
[Why this task matters, any relevant context]

## Objectives
- [ ] Objective 1
- [ ] Objective 2

## Implementation Steps

### Phase 1: [Phase Name]
- [ ] Step 1.1
- [ ] Step 1.2

### Phase 2: [Phase Name]
- [ ] Step 2.1
- [ ] Step 2.2

## Resources Required
- [ ] Resource 1
- [ ] Resource 2

## Approval Required
[ ] Yes - Move to Pending_Approval
[ ] No - Can proceed immediately

## Notes/Considerations
[Any important notes, risks, or dependencies]

---
*Generated by Create Plan Skill*
```

## Example

User: /create-plan Needs_Action/EMAIL_1234.md

Response: "Created plan Plans/PLAN_EMAIL_1234.md for task EMAIL_1234. Task moved to In_Progress."
